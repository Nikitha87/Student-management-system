# Student Management System (Python)

## 📌 Description
This is a simple command-line based Student Management System developed using Python. It allows users to manage student records efficiently.

## 🚀 Features
- Add new student
- View all students
- Update student details
- Delete student records

## 🛠️ Technologies Used
- jupiter Notebook

## ▶️ How to Run
1. Install Python
2. Run the file:
   python main.py

- Implemented CRUD operations
